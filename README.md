# Boti-K Pure — Campaña Digital

Documento interactivo (landing page de página única) para el TP2 de Marketing Digital.

## Cómo publicarlo en GitHub Pages

1. Subí **todo el contenido de esta carpeta** al repositorio (manteniendo la estructura).
2. En el repo: **Settings → Pages → Build and deployment → Source: Deploy from a branch**, elegí la rama (`main`) y carpeta `/root`.
3. Esperá ~1 minuto y abrí la URL que te da GitHub. `index.html` se sirve automáticamente.
